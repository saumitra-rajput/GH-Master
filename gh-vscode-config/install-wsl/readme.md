# WSL Installation 

## Via CMD
- Install: Open CMD as Admin and run: 
    ```
    wsl --install.
    ```

- Process: This enables the Virtual Machine Platform, downloads the Linux kernel, and installs Ubuntu by default.
- Final Step: Restart your PC. After rebooting, you will be prompted to set a Linux username and password.

- WSL has default `Ubuntu distribution`


## Other use commands
- Use  ```wsl.exe --list --online``` to list available distributions

- ```wsl.exe --install <Distro>``` to install.
